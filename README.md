# Amazon Price Tracker (PyQt5 + Selenium)

A simple desktop application built with Python, PyQt5, and Selenium.

This project was created as a learning exercise to practice:

- Building basic GUIs with PyQt5  
- Web scraping with Selenium  
- HTML parsing using BeautifulSoup  
- Saving data locally with CSV files  

The application allows users to track Amazon product prices and receive alerts when a target price is reached.

---

## Features

- Enter an Amazon product URL
- Fetch product title and current price
- Set a target price for alerts
- Save price history to a CSV file
- Optional headless browser mode
- Simple GUI interface

---

## Technologies Used

- Python  
- PyQt5  
- Selenium  
- BeautifulSoup  
- WebDriver Manager  

---

## How to Run

1) Install dependencies:

```
pip install PyQt5 selenium beautifulsoup4 webdriver-manager lxml
```


2) Run the application

```
python price.py
```
